📖 Materi

Conditional Type 1 — Real / Possible

Formula:

If + S + V1 (Simple Present), S + will + V1

Digunakan untuk situasi yang mungkin terjadi di masa depan jika kondisi terpenuhi.

✅ If it rains, I will bring an umbrella.

✅ If you study hard, you will pass the exam.

✅ If I have free time, I will watch anime.

⚠️ Jangan gunakan "will" di bagian IF clause!

Conditional Type 2 — Unreal Present

Formula:

If + S + V2 (Simple Past), S + would + V1

Digunakan untuk situasi tidak nyata / imajinasi di masa sekarang.

✅ If I had wings, I would fly around the world.

✅ If she were rich, she would buy a mansion.

✅ If he got a million followers, he would be famous.

⚠️ Gunakan "were" untuk semua subjek (I were, he were, she were).

Conditional Type 3 — Unreal Past

Formula:

If + S + had + V3, S + would have + V3

Digunakan untuk situasi tidak nyata di masa lalu (penyesalan).

✅ If I had studied, I would have passed.

✅ If they had practiced, they would have won.

✅ If we had arrived on time, we wouldn't have missed it.

⚠️ Type 3 = Past Perfect (had + V3) di IF clause.
